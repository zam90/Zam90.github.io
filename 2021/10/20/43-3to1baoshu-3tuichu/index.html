n人围圈报数问题

更新于2022-04-24|学习笔记
|字数总计:820|阅读时长:3分钟|阅读量:

[TOC]

代码语言

C语言

问题描述

n个人围成一圈，从1-n顺序编号。从第1个人开始循环报数，报到3的人退出圈子(1 2 3 1 2 3…)，问最后留下的人编号是多少？

问题分析

n个人围成一圈报数，最后剩下1个人，所以退出的人数为n-1。

探究一下3人围成一圈报数顺序：第1个人报1，第2个人报2，第3个人报3（3号退出圈子），第1个人报1，第2个人报2，第1个人报3（1号退出圈子）。所以3个人循环报数，最后剩下2号。在这个过程中，需要有一个变量循环访问圈子里的人（第i个人报数），还需要一个变量记录当前报的数（报到了j），同时需要一个一维数组记录第i个人是否在圈子里。

int n = 0;	//人数n
int i = 0;	//循环访问圈子里的人
int j = 0;	//当前报数
int quit = 0;	//退出圈子的人数
int a[999];		//记录第i个人是否在圈子里：a[i]=1表示在圈子中，a[i]=0表示已经退出了圈子

初始化以上变量，将a[n]初值都赋为1（起始的时候n个人都在圈子里），从0-n循环遍历i，直到quit=n-1（退出圈子的人数等于n-1，即只剩1个人）。在循环过程中，每次循环i和j都加1（第i个人报第j个数）；当i循环到第n个人时，将i赋值为0，重新循环遍历；当j=3时，将a[i]置0（该人报到3，退出圈子）。

while (1) {
    //quit=n-1，只剩一个人，退出循环
    if (a[i] == 1 && quit == n - 1) {
        printf("最后1个人编号是%d", i + 1);
        break;
    }
    //报到的不是3，j++
    if (a[i] == 1 && j != 3) {
        j++;
    }
    //报到的是3，该人退出队伍（a[i] = 0），重置j，退出人数quit+1
    if (a[i] == 1 && j == 3) {
        a[i] = 0;
        j = 0;
        quit++;
    }
    i++;
    if (i == n) {
        i = 0;
    }
}

结果运行

代码清单

#include <stdio.h>
#include <stdlib.h>

int main()
{
    int n = 0;	//人数n
    int i = 0;	//循环访问圈子里的人
    int j = 0;	//当前报数
    int quit = 0;	//退出圈子的人数
    int a[999];		//记录第i个人是否在圈子里：a[i]=1表示在圈子中，a[i]=0表示已经退出了圈子

    //将a[n]初值置1，表示一开始所有人都在圈子里
    for (i = 0; i < 999; i++) {
        a[i] = 1;
    }
    printf("问题描述：n个人围成一圈从1到3报数，报到3的人退出圈子，最后留下的人是多少号？\n");
    printf("请输入n：");
    scanf_s("%d", &n);
    i = 0;

    while (1) {
        //quit=n-1，只剩一个人，退出循环
        if (a[i] == 1 && quit == n - 1) {
            printf("最后留下的人编号是%d\n", i + 1);
            break;
        }
        //报到的不是3，j++
        if (a[i] == 1 && j != 3) {
            j++;
        }
        //报到的是3，该人退出队伍（a[i] = 0），重置j，退出人数quit+1
        if (a[i] == 1 && j == 3) {
            a[i] = 0;
            j = 0;
            quit++;
        }
        i++;
        if (i == n) {
            i = 0;
        }
    }
    return 0;
}
