SW智能库存管理系统项目

项目介绍

本项目采用云服务混合架构（B/S+C/S），用户可以通过浏览器访问系统进行库存、人员、通知等相关操作，也可以通过客户端扫描货物二维码快速出入库。

团队成员

张京宇 温蟾圆 吴彩元 张广浩 武文斌

编程语言

HTML ; JavaScript ; PHP ; SQL ; Swift

硬件环境

开发：CPU：Intel i7 主频2.8Ghz 运行内存：16GB 硬盘内存：1.5TB
运行：CPU主频1GHz以上 运行内存：1GB以上 硬盘内存：150MB以上 客户端iPhone4以上机型

软件环境

开发：服务端开发操作系统：Windows 10 客户端开发操作系统：macOS Catalin; 服务端：Dreamweaver+WAMP环境 客户端：Xcode
运行：服务端：Linux 客户端：iOS8.0; 服务端：Apache服务器、MySQL数据库

主要功能

服务端：
登录注册：团队注册、团队登录、个人注册、个人登录
库存管理：产品库存及信息展示、添加产品、手动入库、手动出库、入库记录查看、出库记录查看、数据统计、数据更正、数据删除
员工管理：个人信息管理、人员信息管理、人员权限管理
通知公告：通知展示、通知发布

客户端：
出入库操作：扫码入库、扫码出库
